# Objetivos del Proyecto

## Objetivo general

Desarrollar un sistema de gestión de incidentes en tiempo real y despacho de patrullas, mediante la aplicación de un ciclo de vida de desarrollo de software estructurado que permita la correcta planificación, análisis, diseño, implementación e implantación de la solución.

## Objetivos Específicos

* Realizar el levantamiento de información del sistema mediante la recolección de requerimientos funcionales y no funcionales, identificando actores y casos de uso asociados al proceso de gestión de incidentes y despacho de patrullas.
* Diseñar la arquitectura del sistema, definiendo la estructura de módulos, el modelo de datos y los componentes de integración con servicios de geolocalización.
* Implementar los componentes del sistema de acuerdo con las especificaciones definidas en la etapa de diseño, siguiendo estándares de desarrollo de software.
* Implantar el sistema en un entorno controlado, realizando su despliegue y configuración operativa inicial.
