#### 2. Costo de Inicialización (Puesta en Marcha)
Gastos necesarios para el despliegue del sistema en una sede operativa (ej. Comisaría o Centro de Detención).

* **Infraestructura de Servidores:** * Servidor físico local para persistencia de datos (procesador multinúcleo, 16GB RAM): **$600**
    * Sistemas de alimentación ininterrumpida (UPS): **$120**
* **Periféricos de Captura:** * Micrófonos de alta fidelidad para testimonios y lectores de huella básicos: **$150**
* **Seguridad y Licenciamiento:** * Certificados SSL de validación extendida y configuración de firewalls: **$80**
* **Capacitación de Personal:** * Talleres de inducción para Oficiales y Asesores Legales: **$100**
* **Total Inicialización:** **$1,050**

#### 3. Costo Mensual Operativo (Mantenimiento)
Gastos recurrentes para garantizar que el sistema permanezca activo, seguro e inmutable 24/7.

* **Servicios Cloud / Backup:**
    * Almacenamiento redundante para respaldos cifrados de los testimonios: **$25**
* **Mantenimiento Técnico:**
    * Soporte técnico preventivo y actualizaciones de parches de seguridad: **$100**
* **Conectividad:**
    * Servicio de internet dedicado con IP fija para auditoría remota: **$60**
* **Licenciamiento de APIs:**
    * Uso de servicios externos (opcionales) para verificación o auditoría: **$15**
* **Total Mensual Operativo:** **$200 / mes**

> **Nota de Factibilidad:** El proyecto se considera altamente factible desde el punto de vista económico, ya que la inversión inicial se recupera mediante la reducción de costos procesales derivados de juicios anulados por errores de forma y la optimización del tiempo del personal administrativo.
