# Objetivos

## Objetivo General

Diseñar un módulo de gestión integral para digitalizar el sistema policial, recolectar los detalles de la escena del crimen y automatizar la detección de patrones criminales.

## Objetivos Específicos

- Desarrollar un sistema de registro de hechos delictivos que centralice el registro de información obligatoria para establecer una base de datos controlada mediante roles de acceso diferenciados.

- Mostrar en la interfaz de procesamiento de campo un checklist secuencial que facilite el registro de pruebas del hecho delictivo.

- Programar en el sistema la detección de patrones que permita categorizar los modus operandi y cruzar información de la actividad delictiva.
