# 🔴 Equipo Rojo — Definición Inicial del Sistema

## Descripción General

El equipo Rojo tiene como objetivo el desarrollo de un sistema de registro criminal que permita a investigadores y funcionarios documentar crímenes, denuncias, firmas conductuales y perfiles de sospechosos de manera estructurada y trazable.

## Casos de Uso Principales

### CU-A: Registro de Crímenes y Denuncias
- Un funcionario o investigador puede registrar crímenes y denuncias formales en el sistema.
- Cada registro queda asociado a un número de caso único.
- Se almacenan datos del hecho, fecha, lugar y personas involucradas.

### CU-C: Registro de Firma Conductual
- El sistema permite registrar patrones de comportamiento asociados a delincuentes.
- Facilita la identificación de modus operandi y tendencias criminales.
- Sirve de apoyo para análisis predictivo e investigaciones en curso.

### CU-D: Registro de Sospechosos
- Módulo dedicado al registro de sospechosos vinculados a casos activos.
- Incluye información biográfica, antecedentes y vínculos con otros registros del sistema.
- Permite la gestión de relaciones entre sospechosos, crímenes y evidencias.

## Actores del Sistema

| Actor | Rol |
|-------|-----|
| Funcionario/Investigador | Registra crímenes, denuncias y sospechosos |
| Analista Criminal | Consulta firmas conductuales y patrones |
| Supervisor | Valida y aprueba registros sensibles |

## Notas Técnicas

- Sistema basado en nodos para vincular crímenes, sospechosos y evidencias.
- Búsqueda por patrones conductuales para soporte a investigaciones.
- Trazabilidad completa de cada registro con historial de modificaciones.
